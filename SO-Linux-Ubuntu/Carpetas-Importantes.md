# 📗 Carpetas Importantes que son de Linux

Detallaremos las carpetas y sus caracteristicas de que sirven

### 📘  Raiz (root)

    / 

> Es la Raiz donde empiezas a la ruta

### 📘 Carpeta Home

    /home

> Aca se encuentran los usuarios (se guarda documentos descargas y configuraciones personales)

### 📘 Carpeta etc

    /etc
    
> es donde se configura el sistema (SSH, RED, USUARIOs) Editas el sistema donde se guarda TODO en Archivos de textos configurables.
 
### 📘 Carpeta var

    /var

> Archivos que cambian (jogs y datos) Erroes de sistema y actividad

### 📘 Carpeta bin y usr/bin

    /bin
    /usr/bin

> Programas y comandos (ls, cp, etc) librerias de comandos basicos y programas que se ejecutaron

### 📘 Carpeta tmp

    /temp

> Son archivos Temporales

### 📘 Carpeta root

    /root

> Su trabajo es separa el Admin con el usuario normal

### 📘 Carpeta dev

    /dev

> Dispositivos conectados (discos, usb) y eso por que linus todo es un archivo

### 📘 Carpeta boot

    /boot

> Tiene archivos del sistema operativo

### 📘 Carpeta opt

    /opt

> Son programas opcionales de uuso  de software instalacio manualmente

### 📘 Carpeta lib

    /lib

> Programas que necesita librerias para funcionar correctamente

#### 📒🎄 Recuerda para que empiezes a navegar usas el *cd /*
